## miot/service/security
安全相关服务

**Export**: public  
**Doc_name**: 系统服务_安全模块  
**Doc_index**: 18  
**Example**  
```js
import {Service} from 'miot'

Service.security.shareSecureKey(deviceID, shareUid, {})
 .then(secureKey=>{
    ...
 })

Service.security.loadSecureKeys(deviceID).then(secureKeys=>{
   if(secureKeys.length > 0){
      const key = secureKeys[0];
      key.status = 1;
      key.save().then(key=>{...})
   }
})
```
